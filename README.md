# Start project (production)

## Client

> cd client

> create file .env from example .env.template

> npm i

> npm run build

start file `/build/index.html`

## Server

> cd server

> create file .env from example .env.template

> npm i

> npm run start


# Start project (development)

## Client

> cd client

> create file .env from example .env.template

> npm i

> npm run start

open http://localhost:3000/

## Server

> cd server

> create file .env from example .env.template

> npm i

> npm run start-dev
